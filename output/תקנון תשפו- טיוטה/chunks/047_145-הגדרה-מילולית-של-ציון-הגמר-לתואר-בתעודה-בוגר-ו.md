## **14.5 הגדרה מילולית של ציון הגמר לתואר בתעודה ("בוגר" ו"מוסמך"):**

14.5.1 ציון הגמר לתואר בוגר

רמת הסיום נקבעת לפי מקומו היחסי של התלמיד בין המסיימים בחוג באותה שנה (בתנאי שהציון לא יהיה נמוך מהסף שנקבע לכל רמה), כמפורט להלן:

|  |  |  |  |
| --- | --- | --- | --- |
| רמת סיום | אחוזים | סף ציון מינימלי | |
| בהצטיינות | 20% | | 85 |
| מתוכם- בהצטיינות יתרה | 2% העליונים | | 90 |

* בחוגים בהם מספר המסיימים לא מאפשר חישוב מדרג חוגי יתכן איחוד בין חוגים.
* בממוצע החוגי נכללים כל הקורסים שנלמדו במסגרת תכנית הלימודים בחוג ורק קורסים אלו

14.5.2 ציון הגמר לתואר מוסמך

עבור תואר מוסמך נקבעת רמת הסיום לפי מקומו היחסי של התלמיד בין המסיימים בפקולטה באותה שנה (בתנאי שהציון לא יהיה נמוך מהסף שנקבע לכל רמה), כמפורט להלן:

|  |  |  |  |
| --- | --- | --- | --- |
| רמת סיום | אחוזים | סף ציון מינימלי | |
| בהצטיינות | 25% | | 85 |
| מתוכם- בהצטיינות יתרה | 3% עליונים | | 92 |
